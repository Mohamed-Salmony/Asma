# ASMA for Advanced Technologies

This repository contains the Next.js 15 migrated codebase for ASMA Advanced Technology.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm.cmd install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm.cmd run dev`
